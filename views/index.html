Welcome to SimpleStore

Simplestore is an online file storage where you can save lightweight files temporarily. This website is hugely inspired by protectedtext.com eventhough it's still not secure as protectedtext. If you want to create a new storage just enter add "/yourstorage" after our domain or you can use the input box below

Features & Future Update(Maybe)

  1. Every files that's uploaded has lifespan for 1 hour
  2. Security will be updated soon
  3. Better UI in the future
  4. Better Authentication (Without session)
  5. Adding text-editor feature
  6. Using database
